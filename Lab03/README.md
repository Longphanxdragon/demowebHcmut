# Lab03 - PHP, MySQL, AJAX Examples

This is the implementation of Lab03 requirements, demonstrating basic PHP, database interaction, forms, and AJAX.

## Features Implemented

1. **Forms Examples** (forms.php)
   - GET method form
   - POST method form
   - File upload form

2. **Database Connection** (database.php)
   - PDO connection to SQLite database
   - Display products from database

3. **AJAX Examples** (ajax.php)
   - Basic XMLHttpRequest AJAX
   - jQuery AJAX

4. **Simple Shop Application** (shop.php)
   - Display products
   - Add new products via AJAX
   - Delete products via AJAX

## Database

Uses SQLite database (shop.db) with a products table.

## How to Run

1. Start PHP server: `php -S localhost:8080`
2. Open http://localhost:8080 in browser
